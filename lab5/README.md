## Опис реалізованого функціоналу

У рамках лабораторної роботи було розроблено мобільний застосунок з використанням сучасної file-based маршрутизації (Expo Router). Додаток демонструє архітектуру з публічною та захищеною зонами.

**Основні реалізовані можливості:**

1. **Глобальний стан (Context API):** Створено `AuthContext` для управління станом авторизації користувача (логін, реєстрація, вихід).
2. **Публічні екрани (Група `auth`):** Реалізовано екрани входу (`login.tsx`) та реєстрації (`register.tsx`) з відповідними формами.
3. **Захищена маршрутизація (Група `app`):** Створено `_layout.tsx`, який перевіряє стан `isAuthenticated` і виконує перенаправлення неавторизованих користувачів на екран логіну.
4. **Головний екран (Каталог):** Відображення списку товарів (FlatList) з можливістю переходу до деталей.
5. **Динамічні маршрути:** Реалізовано екран `details/[id].tsx`, який отримує ідентифікатор товару з URL та відображає розширену інформацію.
6. **Обробка помилок (404):** Додано екран `+not-found.tsx` для обробки неіснуючих маршрутів.
