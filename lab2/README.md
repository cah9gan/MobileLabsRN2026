## Опис проєкту

Цей проєкт демонструє побудову вкладеної архітектури навігації у React Native та методи оптимізації великих обсягів даних за допомогою віртуалізованих списків.

Додаток базується на `Drawer Navigator` (бокове меню), який містить:

1. **Екран "Новини":** Використовує вкладений `Stack Navigator`. Головна сторінка відображає стрічку новин за допомогою компонента `FlatList`. Реалізовано динамічне підвантаження даних (Infinite Scroll) та оновлення списку жестом (Pull-to-Refresh). При натисканні на новину відбувається перехід на екран деталей з передачею параметрів.
   <img src="/lab2/pictures/scrollNewsPage.png" width="150" />
   <img src="/lab2/pictures/newsPage.png" width="150" />
2. **Екран "Контакти":** Демонстрація групованих даних за допомогою компонента `SectionList` (Співмешканці, Координатори тощо).
   <img src="/lab2/pictures/contactsPage.png" width="150" />
3. **Кастомне Drawer Menu:** Бокове меню кастомізовано: додано аватар, ПІБ та групу студента над стандартними пунктами навігації.
   <img src="/lab2/pictures/profilePage.png" width="150" />

## Інструкція із запуску

1. Перейти в директорію другої лабораторної роботи:
   ```bash
   cd MobileLabsRN2026/lab2
   ```
2. Встановити залежності:
   ```bash
   npm install
   ```
3. Запустити сервер Expo (рекомендується з очищенням кешу для коректної роботи анімацій меню):
   ```bash
   npx expo start -c
   ```
