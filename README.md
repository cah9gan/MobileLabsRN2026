## Опис проєкту

Цей проєкт є найпростішим мобільним додатком, створеним за допомогою React Native та Expo.
Додаток містить нижню навігацію (Bottom Tabs) та складається з трьох основних екранів:

1. **Головна (Новини):** Стрічка новин з використанням компонента `ScrollView`.
   ![Головна](path/to/home_screenshot.png)
2. **Фотогалерея:** Демонстрація роботи зі списком у вигляді сітки (2 колонки).
   ![Галерея](path/to/gallery_screenshot.png)
3. **Профіль:** Форма реєстрації користувача з використанням базових компонентів вводу (`TextInput`).
   ![Профіль](path/to/profile_screenshot.png)

У додатку реалізовано глобальні Header (з логотипом університету) та Footer.

## Інструкція із запуску

1. Клонувати репозиторій:
   ```bash
   git clone [https://github.com/ТВІЙ_ЛОГІН/MobileLabsRN2026.git](https://github.com/ТВІЙ_ЛОГІН/MobileLabsRN2026.git)
   ```
2. Перейти в директорію проєкту:
   cd MobileLabsRN2026/lab1
3. Встановити залежності:
   npm install
4. Запустити сервер Expo:
   npx expo start

## Опис основних способів запуску мобільного додатка

[cite_start]Під час розробки та тестування React Native застосунків через Expo існують два основні способи запуску:

### 1. Запуск на фізичному пристрої (через Expo Go)

- **Призначення:** Швидке тестування на реальному "залізі".
- **Особливості:** Потребує встановленого додатку **Expo Go** на смартфоні (iOS/Android). ПК та смартфон мають бути підключені до однієї Wi-Fi мережі (або використовується прапорець `--tunnel`). Запуск відбувається шляхом сканування QR-коду з терміналу.
- **Відмінності:** Максимально наближений до реального користувацького досвіду (робота сенсора, реальна продуктивність). Не навантажує оперативну пам'ять комп'ютера.

### 2. Запуск на емуляторі (Android Virtual Device / iOS Simulator)

- **Призначення:** Дебаггінг та тестування на різних розмірах екранів без необхідності мати фізичні пристрої.
- **Особливості:** Потребує встановлення та налаштування **Android Studio** або **Xcode**. Запускається клавішею `a` (для Android) або `i` (для iOS) у терміналі Expo.
- **Відмінності:** Споживає значні ресурси комп'ютера (RAM, CPU). Дозволяє зручно імітувати зміну геолокації, рівня заряду батареї та інших системних параметрів.
