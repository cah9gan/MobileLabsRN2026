Кожна директорія містить окремий ізольований проєкт та власну детальну документацію (інструкції із запуску, опис функціоналу та скріншоти роботи застосунку). Натисніть на посилання нижче, щоб перейти до відповідної роботи:

  _Створення базових екранів, робота з компонентами ScrollView, TextInput та формами._

- 🧭 **<a href="./lab2/README.md">Лабораторна робота №2: Вкладена навігація та оптимізація списків</a>**
  _Побудова архітектури за допомогою Drawer та Stack навігаторів. Віртуалізація великих обсягів даних через FlatList та SectionList._

- 👆 **<a href="./lab3/README.md">Лабораторна робота №3: Використання кастомних жестів та стилізація інтерфейсу</a>**
  _Реалізація гри-клікера з обробкою мультитач-жестів (Tap, Pan, Pinch, Fling) за допомогою React Native Gesture Handler та Reanimated._

---

### 🛠 Технологічний стек

- **Core:** React Native, Expo, TypeScript
- **Navigation:** React Navigation (Bottom Tabs, Drawer, Stack), Expo Router (File-based routing)
- **State Management:** React Context API
- **Gestures & Animations:** React Native Gesture Handler, React Native Reanimated
- **Hardware APIs:** Expo Camera, Expo Location, Expo Media Library, React Native Maps
- **Styling:** NativeWind (Tailwind CSS), StyleSheet
