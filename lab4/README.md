## Опис реалізованого функціоналу

У рамках лабораторної роботи було розроблено мобільний додаток, який взаємодіє з апаратними можливостями пристрою та сторонніми API.

**Основні реалізовані можливості:**

1. **Камера та Галерея (`expo-camera`, `expo-image-picker`):**
   - Реалізовано запит на дозволи використання камери та медіа-бібліотеки.
   - Можливість створення фотографій за допомогою компонента `CameraView`.
   - Збереження зроблених знімків у файлову систему (галерею) пристрою.
   - Вибір існуючих фотографій з галереї та їх відображення в інтерфейсі.
2. **Геолокація та Карти (`expo-location`, `react-native-maps`):**
   - Динамічне отримання поточних GPS-координат (широта та довгота) пристрою.
   - Інтеграція інтерактивної карти, яка автоматично центрується на отриманих координатах.
   - Встановлення маркеру (`Marker`) на місці поточного перебування користувача.
3. **Стилізація:**
   - Використано бібліотеку NativeWind для стилізації компонентів.
   - Реалізовано базову підтримку адаптації під світлу та темну теми системи.

## Інструкція із запуску

Для повноцінного тестування функціоналу застосунок **обов'язково** потрібно запускати на фізичному мобільному пристрої через додаток Expo Go, оскільки веб-браузер не підтримує API камери та нативних карт.

1. Перейти в директорію проєкту:
   ```bash
   cd MobileLabsRN2026/lab4
   ```
2. Встановити залежності:
   ```bash
   npm install
   ```
3. Відсканувати QR-код за допомогою Expo Go.
