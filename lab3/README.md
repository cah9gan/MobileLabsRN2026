## Опис реалізованого функціоналу

У рамках лабораторної роботи було розроблено мобільний додаток "Gesture Clicker", який демонструє роботу з кастомними жестами користувача, складними анімаціями та глобальним станом.

**Основні реалізовані можливості:**

1. **Обробка жестів (React Native Gesture Handler):**
   - **Tap (Одинарний та подвійний клік):** Отримання базових очок з пружинною анімацією (spring) зменшення/збільшення об'єкта. Використано `Gesture.Exclusive` для розділення логіки одинарного та подвійного натискання.
   - **Long Press (Довге натискання):** Нарахування бонусних очок при утриманні елемента понад 1 секунду.
   - **Pan (Перетягування):** Можливість вільного переміщення об'єкта по екрану з автоматичним поверненням у центр після відпускання.
   - **Fling (Свайп):** Розпізнавання різких жестів вліво та вправо для нарахування випадкової кількості очок.
   - **Pinch (Масштабування):** Зміна розміру об'єкта двома пальцями.
2. **Анімації (React Native Reanimated):**
   Усі трансформації (scale, translateX, translateY) реалізовані через `useSharedValue` та `useAnimatedStyle`, що забезпечує плавну роботу в UI-потоці при 60 FPS.
3. **Глобальний стан (Context API):**
   Реалізовано `GameContext` для збереження статистики жестів та передачі даних між екраном клікера та екраном завдань (Challenges).
4. **Стилізація та Теми:**
   Інтерфейс побудовано з використанням бібліотеки **NativeWind** (Tailwind CSS для React Native). Реалізовано підтримку світлої та темної теми з миттєвим перемиканням на екрані налаштувань.

## Інструкція із запуску

Оскільки додаток активно використовує нативні модулі для обробки мультитач-жестів та анімацій, **настійно рекомендується тестувати його на фізичному мобільному пристрої**. У веб-браузері жести на кшталт Pinch (масштабування) працювати не будуть.

1. Перейти в директорію проєкту:
   ```bash
   cd MobileLabsRN2026/lab3
   ```
2. Встановити всі залежності:
   ```bash
   npm install
   ```
3. Запустити сервер Expo з очищенням кешу бандлера (обов'язково після налаштування Babel-плагінів для Reanimated та NativeWind):
   ```bash
   npx expo start -c
   ```
